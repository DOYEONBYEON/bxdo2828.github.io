Build Process


1.Hydure 테마 적용

2. Git & Github 관련 설명 포스트 추가

3.Disqus를 이용하여 댓글 기능 추가

4. Google Analytics기능 추가
(블로그의 이용자 통계 추적 서비스)

5. 사진 포스팅

6 .태그 기능 추가


우선 수업을 따라가며 Hydure 테마를 적용하였고
인터넷 자료를 통해 Git 과 Github에 대한 내용을 포스팅,
수업에 나온 Disqus를 이용하여 댓글 기능을 추가하였다.
그리고 Google Anaytic를 통해 블로그 이용자 통계 추적할 수 있도록 하였고
사진을 함께 포스팅 하였으며 태그 기능도 넣었다.
